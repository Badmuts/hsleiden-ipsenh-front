# IPSENH Front
Dit is een React frontend voor het IPSENH project.

## Getting started

```sh
# Start react met server at localhost:9000
$ npm start
# Or when using yarn
$ yarn start 

# Run tests
$ npm test
# Or when using yarn
$ yarn test
```

For more info about how to use this react app and its tools check [REACT.md](REACT.md)
